- **Build plugin assets**: npm run build completes without new warnings or bundle errors.

- **Edit a storymap with CKEditor**: The storymap editor loads, inline rich text editing works, and image upload still attaches media.
- **Reorder layers and storymap slides**: Drag-and-drop or movable ordering works in all UIs that currently rely on reordering.
- **Edit layer and legend forms**: Schema-driven forms, legend settings and interaction settings still render and save.
- **Filter stories by date in Discovery**: The date range picker applies and clears filters correctly.
